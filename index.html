HTML構成の概略

HTMLでページを作るための基本的な構成を解説します。以下の情報は必須情報です。
・「!DOCTYPE HTML」でHTMLのバージョンを宣言します。HTMLのバージョンによって記述が違います。
・headタグには、ページのメタ情報（ページの設定やインターネット検索で使用される情報）を設定します。
・bodyタグの中には、ページに表示される内容を記述します。

HTML 

<!DOCTYPE HTML><!-- HTML5のバージョンを宣言 -->
<html><!-- HTMLの最上位要素 -->
    <head>
        <!-- ページのメタ情報をheadタグ内に記述する -->
        <title>ページのタイトル</title>
    </head>
    <body>
        <!-- ページの内容をbodyタグ内に記述する -->
    </body>
</html>

HTMLは文書構造を定義するための書式です。その画面で表示する情報の順番や構成を決めるために存在しています。 そのため、HTML側では（style属性の指定で設定することはできますが）レイアウトや見た目のスタイルを設定しません。 スタイルの設定は「スタイルシート」（.cssファイル）で行なうようにしてください。
以下のようにheadタグにlinkタグでスタイルシートのパスを設定します。

HTML 

<!DOCTYPE HTML>
<html>
    <head>
        <title>ページのタイトル</title>
        <link rel="stylesheet" href="./destyle.css">
        <link rel="stylesheet" href="./style.css">
    </head>
    <body>

    </body>
</html>

セクション系HTMLタグ

セクション系HTMLタグは、コンテンツの構造を表現するために使用します。主にWeb検索エンジンに認識されやすくするためのSEO対策で使用されます。社内などの限らてたローカルネットワーク上のWebアプリケーションでは、あまり意識する必要はないと思います。

h1～h6要素

文章の段落をあらわす要素

section要素

文章の段落をあらわす要素

main要素

文章の段落をあらわす要素

HTML 

<main>
  <h1>見出しA</h1>
  <h2>見出しB</h2>
  <section>
    <h3>見出しC</h3>
  </section>
</main>

グローバル属性

属性とは、要素にすべての要素に共通して指定できる属性のことを

id属性

要素に固有の名前を付けるための属性

class属性

要素が属する種類・分類をあらわすための属性

style属性

CSSの宣言を設定するための属性

tabindex属性

Tabキーによる移動の順序を設定する属性

hidden属性

要素を表示しないための属性

HTML 

<input type="text" id="textbox" class="text-box" style="width: 300px;" tabindex="-1" placeholder="テキストを入力してください" value="こんにちわ！！！" />
